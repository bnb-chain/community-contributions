# BNB Chain Grant Wishlist: Most demanding and innovative ideas

The BNB Chain Grant Wishlist is a program that aims to support and reward the most ambitious and creative projects that leverage the BNB Chain ecosystem. The program invites developers, entrepreneurs, researchers, and enthusiasts to submit their proposals for novel and impactful solutions that can benefit from the fast, secure, and scalable BNB Chain ecosystem. The program offers grants, as well as technical guidance, mentorship, and exposure to the BNB Chain community.
